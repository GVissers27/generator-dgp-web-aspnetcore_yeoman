Welcome


This is a very small sample page that uses acpaas ui components and styles which you can use as a starting point to build your own page.
This page comes in three flavor: ACPaaS, Digipolis and Antwerp.
To switch between the three:

  1. See styles.scss in the styles folder.
  2. There are some images that can be used as favicon and logos in the assets/images folder.
  3. Take one of the above or anotther logo and make sure to change the refernce in app.component.html.
  4. Should the footer say Digipolis, Stad Antwerpen or something else ?
Once you decide on which style to use you can remove the part you dont need.

You can add your own non-component-specific styles in the styles/main.scss file.
